# Feature: Task CRUD

## Operations
- **Create**: Add a task with title and optional description
- **Read**: List all tasks for the current user
- **Update**: Edit task title and/or description
- **Delete**: Remove a task
- **Toggle Complete**: Mark a task as done/undone

## Console (Phase I)
- Interactive menu with numbered options
- In-memory storage (data lost on exit)

## Web App (Phase II)
- REST API endpoints with JWT auth
- Real-time UI updates after each operation
- Edit modal for updating tasks
