# Todo App - Overview

A full-stack todo application built in two phases:

1. **Phase I** - Python console app with in-memory storage and rich CLI interface
2. **Phase II** - Web app with Next.js frontend and FastAPI backend, JWT authentication, and SQLite database

## Tech Stack

- **Console**: Python 3.13, rich
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS
- **Backend**: FastAPI, SQLModel, SQLite, JWT auth
- **Auth**: Custom JWT (email/password)
